📂 Library Paths

Point Reel at one or more folders. Add as many drives/paths as you like — the indexer crawls them all and merges everything into one library. Save, then Rescan.

👥 Profiles

Each profile is a separate login with its own favorites, playlists and history — they never mix. Add one for another person; they sign in with their own 4-digit code at the normal passcode screen.

🔑 Guest Access

Share temporary access to the stream. Each code uses the front passcode box and auto-expires — the guest is kicked the moment it lapses or you revoke it.
